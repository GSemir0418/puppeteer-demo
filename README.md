# 安装

跳过 chrome 下载

```bash
export PUPPETEER_SKIP_DOWNLOAD=true
npm install puppeteer
```

# 使用

爬取掘金网站，关于前端面试的文章，按时间排序

```bash
node reptile.js
```

屏幕截图

```bash
node screen-shot.js
```

屏幕导出 pdf

```bash
node screen-pdf.js
```
